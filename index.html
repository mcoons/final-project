Learning GameMaker Studio Scripting

May 12, 2018

Topic

Learn some scripting for Gamemaker Studio 2 for the Game-hack-a-thon some of us are competing in. My assignment is to learn how to program moving platforms for our levels.

Set the Stage

I closed all applications except this and the GameMaker Studio 2 application. I then proceeded to start though the in-app help system and tutorials to familiar myself with the GameMaker scripting language (GML).

Setting the Stage

Reflection

The team created a git repo for our project and after cloning it, I copied it to a play directory where I could mess around with learning and not bombard the real repo with all my playing.

I implemented moving platforms three different ways, but due to other code we were trying to interface with, I had glitches that were hard to deal with.Playing around and trying to implement the platforms in different ways helped me to learn different facets of the language and gaming engine. Had I stuck with one implementation, I would not have learned as much about the gaming engine.

Minimizing the applications that were open during 'setting the stage' made it easier to navigate between the GameMaker app as well as the Chrome tabs I had opened for research. I did miss a few slack notifications concerning the project though.

Hopefully our team can complete a level or two and turn it in before the 3 day time period is over. We may want to enter more game-hack-a-thons in the near future once we get use to the software.

Learning DOM Events

May 15, 2018

Topic

I hope to better understand DOM events so I can more easily integrate them into my applications.

Set the Stage

I closed all apps except my music and then I opened Chrome to access my notes and the assignment. I then proceeded to read the heroku links.

Setting the Stage

Reflection

During 'Setting the Stage' I ended up closing Chrome and losing some tabs that I had previously looked up so I had to find them and open them again. This slowed me down some but not too much. I usually don't have too much open that will cause distractions.

As I read about the different document and mouse events I tried to find ways to incorporate some of them into my Pixel Art Maker app. This constituted my "play" portion of my learning. As I incorporated new options in my app I encountered issues that required me to research other events to help fix the issues.

For me, I think that actually learning these events with an app helped me to understand and remember them better than using simple example scripts or repls that only cover a small idea.

I hope that in the future I will be able to incorporate events into my future apps with ease as well as have a better understanding of the types of events available to me.

Learning How to Create a Good LinkedIn Profile

May 16, 2018

Topic

I hope to learn different techniques to creating LinkedIn profiles that catch the eye of an employer.

Set the Stage

I closed all apps except my music and then I used Chrome to Google "creating linkedin profiles". I also looked over some links that Career Service listed.

Setting the Stage

I find that setting my stage can have its pros and cons. While it does help to remove "some" clutter, I usually keep my workspace to only the pertinent windows. I am not much into social apps so closing something like the slack window, while seemingly a good idea, results in missing some messages.

I also find that I have to research previously found material because I accidentally closed a window before creating a bookmark.

Reflection

While I did learn some techniques for creating a good profile, it seems since there are so many different ways to represent yourself, there is no "best" way. I also noticed that the techniques you use and the way you represent yourself could depend on what type of job you are applying for.

I also did this before we met with Career Services. Once we met, they solidified some of the ideas I had come to.

Firebase - 10 Minutes to Deployment

Date: May 2, 2018

Location: Slalom Office, Denver, CO

People I met

  • Adam Jacaruso - Slalom - Solution Architect
  • Siegmund Nagel - Slalom - Consultant

What I Learned

I learned that Firebase is a decent service for quick deployment. I asked Adam about the authentication options and found out that you can integrate authentication through other apps like facebook as well as having general username and passwords.

I also learned that Googles Firebase services have other options you can incorporate other than hosting and authentication, like Cloud Storage, a database server, performance monitoring and a test lab.

Questions I Have

I would be interested in knowing to what extent I could use Google services like Firebase to deploy simple apps and how popular your app would have to be before you hit their limits and actually had to start paying for quota increases.

I would also like to spend some time looking into their test lab to see what it has to offer.

Colorado Independent Game Developers Association

Date: May 16th, 2018

Location: Roostercat Coffee House

People I Met

  • Ash Blue - Organizer - Game Developer
  • Nicky Britt - Possible Mobile - Developer

What I Learned

Talking with some game developers and web developers that were researching game development, I found out that Git and Github were used by a lot of them. They also mentioned using Agile development techniques which strengthened what we are being taught here at Galvanize.

They also talked about another software development management solution called JIRA. It provides bug tracking, issue tracking, and project management functions.

We also talked about other sites and software that could be handy during development like Bit Bucket, Git Lab, Source Tree, Trello, Behavior Designer and Playmaker.

The main software development package most people seemed to mention was Unity 3D. This was good to hear because I already have some familiarity with that IDE.

Questions I Have

I would like to know the average salary of junior game developers. I think that it would be harder finding a job as a game developer but it may be a nice side gig.

It also seems that a lot of them were also or had been web developers so I wonder what type of statistics exist concerning taht connection.
